
                                                   Apache Kafka 
                                    ==========================================

1.What is Apache Kafka ?

  Apache kafka is a communication system that helps different parts of a computer system exchange data by publishing and
  subscribing to topics .  
  
2. Why use kafka ?

   a. High Throughput -> Means, it exchange vast amount of data in small time .
   b. Fault Tolerance (Replication) -> Means it create some copy of data , if one copy is 
                                       deleted , then anathor copy is work and it give the data 
                                       to the user without any exception .
   c. Durable   ->Means , it contains data in long time .
   d. Scalable .
